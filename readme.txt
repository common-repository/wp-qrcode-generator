== Description ==
QRCode Generator - plugin you can generate wide custom and automatic site page URL QR code generator and use them for Elementor, widgets, and Shortcodes. The plugin does not depend on any external services, QR Codes are generated using JavaScript. 

* By: MD Masud Sikdar [https://www.mdmasudsikdar.com](https://www.mdmasudsikdar.com)
* Email: contact@mdmasudsikdar.com

== Features ==
* Required Patterns that support dot style.
* Support for Quiet Zone settings.
* Support custom Position Pattern inner fill and outer border color 
* Support custom Alignment Pattern inner fill and outer border color 
* Support custom Timing Patterns vertical, horizontal color 
* Support Logo images (including transparent PNG images) 
* Support Background Image

= QR Code Generator helps to generate code =
* Custom text/custom link. 
* Current page URL. 
* Mail to link/Email Message. 
* Calendar Event. 
* Contact Information. 
* Google Location. 
* Phone Number. 
* Phone SMS. 
* Skype Contact 
* WIFI Access 

= Styling Options =
* Custom Logo 
* Custom background 
* Border color, width, height 
* Patten Outer / Inner Color 
* Aligment Outer / Inner Color 
* Custom Timing Style 
* Custom Dot Style

== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/qrcode-wp/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
